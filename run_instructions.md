# PxCam 起動・開発用コマンドメモ

PxCamをターミナルから起動して開発・実行するための手順とコマンドの一覧です。

## 1. 準備 (初回のみ)
プロジェクトのルートディレクトリに移動し、必要に応じて依存関係をインストールします。

```bash
# プロジェクトのディレクトリに移動
cd /Users/kenjikishi/projects/PxCam

# パッケージ（Viteなど）をインストール
npm install
```

## 2. 開発サーバーの起動 (通常はこちらを使用)
ローカルでリアルタイムにプレビューしながら開発を行うためのコマンドです。

```bash
npm run dev
```

* **起動後の接続先**: `http://localhost:5173/` (デフォルト)
* **特長**: ソースコード（JS, HTML, CSS）を変更すると、ブラウザが自動的にリロード（ホットリロード）されて変更が即座に反映されます。

## 3. その他のコマンド

| コマンド | 説明 |
| :--- | :--- |
| `npm run build` | 本番環境向けに最適化された静的ファイルを `dist/` ディレクトリに生成します。 |
| `npm run preview` | ビルドした静的ファイルをローカルサーバーで起動し、本番動作のプレビューができます。 |
