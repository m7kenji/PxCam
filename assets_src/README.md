# Source Assets / 編集用ソースアセット

このフォルダは、テクスチャーパターンや各種アセットの編集用元データ（PSD、Aseprite、Illustratorファイルなど）を保管するための場所です。

## なぜこの場所にあるのか？
* `public/` フォルダ以下に編集用データを置くと、本番ビルド（`npm run build`）時に不要な大容量ファイルとして書き出しフォルダ（`dist/`）にコピーされてしまいます。
* このフォルダ（`assets_src/`）に元データを置いて管理し、書き出した軽量な **PNGファイルのみ** を `public/patterns/` に配置することで、本番用のアプリサイズを軽量に保つことができます。
